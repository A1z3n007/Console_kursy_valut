# KZT Price Console (C#)

Консольное приложение на C# (.NET 8), которое выводит цену в тенге (KZT) по текущему курсу онлайн.

## Источник курсов
Используется open endpoint ExchangeRate-API:
https://open.er-api.com/v6/latest/USD
(а также другие валюты по аналогии)

## Запуск
```bash
dotnet restore
dotnet run

## Возможности

Конвертация суммы из любой валюты (USD/EUR/RUB и т.д.) в KZT по текущему курсу

Мини-магазин: товары в разных валютах, вывод итоговой цены в KZT
